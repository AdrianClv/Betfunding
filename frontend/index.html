FAQ

How can I create a project?

You have to go to the "New project" section and fill out the form. Make sure that you have enough ether to pay for the gas of the transaction.
The judge is a trusted third party that is going to verify the project. Due to limitations in the platform, the description have to be written in another web page, and then, put the link to the description.

How can I fund a project?

To fund a project go to the "Projects" section and click on the project name you want to fund, or put its ID in the search box at the top of the page.
Then, go to the project info, write the amount you want to contribute to the project where it says "Bet" and click on "Bad bet".

How can I become a creator?

First of all check if there is already people working on that project (number of nice bets). If there is nobody working on that project and you want to start working on it, you have to send a deposit as a nice bet. If there is already people working on it, then you will have to share the bounty.
The more you send as a deposit and the sooner you start working on the project, the more you will get of the bounty.

About Betfunding

Betfunding is a distributed crowdfunding platform that uses bets instead of donations to support artistic production and creative work. The platform is implemented as a smart contract on Ethereum blockchain, using ether cryptocurrency as the monetary system with no country restrictions.
Unlike other crowdfunding platforms, in Betfunding the investors are the ones that suggest a project and look for creators by the creation of a bounty for their work.
